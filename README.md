# Batu Gunting Kertas AI

Proyek ini adalah permainan sederhana **Batu Gunting Kertas** berbasis web dengan tambahan elemen AI. Pemain dapat bermain melawan komputer yang menggunakan logika AI untuk memilih gerakan secara cerdas.

## Fitur
- Pilihan untuk bermain **Batu**, **Gunting**, atau **Kertas**.
- Komputer menggunakan algoritma untuk memilih gerakan terbaik.
- Hasil permainan ditampilkan secara langsung (Menang, Kalah, atau Seri).
- Antarmuka pengguna yang menarik dan mudah digunakan.

## Teknologi yang Digunakan
- **HTML5**: Untuk struktur dasar halaman web.
- **CSS3**: Untuk desain dan tampilan permainan.
- **JavaScript**: Untuk logika permainan dan integrasi AI.
